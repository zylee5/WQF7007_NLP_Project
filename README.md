# WQF7007 NLP Project

This project studies transfer learning for tweet toxicity/hate classification:
- Stage 1: pretrain a custom BiLSTM on Sentiment140 (binary sentiment).
- Stage 2: transfer and fine-tune on HateXplain (3 classes).
- BERT fine-tuning on HateXplain.

## Main Notebook

- `jupyter-notebook/bilstm_transfer_sentiment140_hatexplain.ipynb`
